DÉPANNAGE:
═══════════════════════════════════════════════════════════════════════════

Si erreur de connexion BD:
  1. Vérifier que MySQL est démarré
  2. Vérifier les identifiants dans config.php
  3. Vérifier que la BD "projetr" existe
  4. Vérifier les privilèges utilisateur root

Si CSS/JS ne se charge pas:
  1. Vérifier les chemins relatifs
  2. Vider le cache navigateur
  3. Vérifier permissions fichiers

Si erreur PHP:
  1. Activer display_errors en dev
  2. Vérifier logs Apache
  3. Vérifier syntaxe PHP avec php -l

Si erreur JavaScript:
  1. Ouvrir console navigateur (F12)
  2. Regarder les erreurs AJAX
  3. Vérifier réponse serveur
