🛠️ ENDPOINTS AJAX DISPONIBLES
═══════════════════════════════════════════════════════════════════════════

POST /controller/reclamation_controller.php
  ├── action=creer              → Créer une réclamation
  ├── action=repondre           → Ajouter une réponse
  ├── action=update_statut      → Mettre à jour le statut
  └── action=supprimer          → Supprimer une réclamation

GET /controller/reclamation_controller.php
  ├── action=lister             → Récupérer toutes les réclamations
  ├── action=detail&id=N        → Récupérer détails + réponses
  └── action=stats              → Récupérer statistiques

🚀 PRÊT À DÉMARRER
═══════════════════════════════════════════════════════════════════════════

ÉTAPES RAPIDES:

1️⃣ Démarrer XAMPP
   - Apache: Start
   - MySQL: Start

2️⃣ Créer la base de données
   - Aller à: http://localhost/phpmyadmin
   - Créer BD: "projetr"
   - Importer: setup.sql

3️⃣ Accéder à l'application
   - Accueil: http://localhost/projetweb/
   - Client: http://localhost/projetweb/view/frontoffice/
   - Admin:  http://localhost/projetweb/view/backoffice/

4️⃣ Tester les fonctionnalités
   - Créer une réclamation
   - Voir dans l'admin
   - Ajouter une réponse
